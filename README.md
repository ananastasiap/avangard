# Авангард и его представители

Проект подготовлен для Краснодарского музыкального колледжа им.Николая Андреевича Римского-Корсакова.

## Установка

```bash
npm install
```

## Скрипты

Запускает dev-сервер Vite.

```bash
npm run dev
```

Собирает билд проекта.

```bash
npm run build
```

Проверяет код проекта линтером.

```bash
npm run lint
```

Запускает локальный preview production-сборки.

```bash
npm run preview
```
